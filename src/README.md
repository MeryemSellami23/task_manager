# 🗂 React Task Manager

A simple and modern Task Manager built with React.  
Supports adding, editing, deleting tasks, search, dark mode, and local storage persistence.

---

## 🚀 Live Demo

👉 https://task-manager-eight-swart.vercel.app/

---

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit tasks
- ❌ Delete tasks
- 🔍 Search tasks
- 🌙 Dark / Light mode
- 📊 Progress tracking
- 💾 LocalStorage persistence (data stays after refresh)
- 📅 Created & completed date tracking
- 🎯 Priority system (Low / Medium / High)

---

## 🛠️ Tech Stack

- React (useState, useEffect)
- JavaScript (ES6)
- CSS3
- LocalStorage
- Vercel (Deployment)
